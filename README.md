# React + Tailwind landing page

## Run locally

```bash
cd landing-page
npm install
npm run dev
```

## What’s inside

- React + Vite + TypeScript
- TailwindCSS (with small reusable utility classes in `src/styles.css`)
- A modern landing page in `src/App.tsx` (hero, features, testimonials, pricing, FAQ, CTA, footer)
